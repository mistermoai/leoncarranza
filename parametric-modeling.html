Parametric Modeling Tools

This page includes documentation and download links for the parametric modeling scripts I made for my Spring 2022 Integrative Design Studio.
3D Furniture
Window Frame

1.

  • - Set up a layer corresponding to the panel connection in grasshopper's dynamic geometry tool.
  • *I recommend using a consistent color for these "baseline" layers. I personally use a shade of pink.

2.

  • - Draw a polyline on this created layer, outlining desired placement for window frame
  • *make sure that all vertices lie on the same "plane" in order to create a planar window

3.

  • - With the grasshopper preview on, you can freely adjust the vertices within the frame

4.

  • - You can select the baseline in rhino and use the "flip" command to change the direction the frame opens towards.

Download 3dfurniture.gh
